OpenHaldex-S3

Modular AWD controller and diagnostics interface for Haldex-equipped platforms. This ui-dev branch is a private development shell for next-generation UX and setup workflow.

  • Firmware target: ESP32-S3 (LilyGo T-2CAN)
  • Core focus: deterministic CAN control + modular signal mapping
  • UI focus: safer setup flow, clearer diagnostics, cleaner production-ready UX
